Plugin AI Assistant for WordPress plugins. Supercharge your plugin management with AI.

== Description ==

Ploogins - AI Assistant helps you manage, install, and activate plugins with the power of AI. Save time, discover new plugins, and automate plugin tasks directly from your WordPress dashboard.

* Effortless plugin management
* AI-powered plugin recommendations
* One-click install and activation

The Plugin calls an external public API at https://api.ploogins.com/public/search. The plugin sends the user query while searching for plugins to get recommended plugins using AI.
